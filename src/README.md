# 🤖 API Automation Bot

A Python-based automation project that fetches real-time data from a public API and displays structured output in the console.

---

## 🚀 Features
- Fetches live data from Agify API  
- Parses JSON response  
- Displays formatted output  
- Beginner-friendly API integration project  

---

## 🛠️ Tech Stack
- Python 3  
- Requests library  
- REST API (Agify)

---

## ⚙️ How It Works
1. Sends a GET request to API endpoint  
2. Receives JSON response  
3. Extracts required data  
4. Prints clean output in console  

---
## 📦 Installation

```bash

pip install -r requirements.txt
```
